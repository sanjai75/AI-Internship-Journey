# Bonus Challenge: Learning with the Feynman Technique

## Topic Chosen: APIs

### Prompt Used

Teach me APIs using the Feynman Technique.

---

## AI Explanation

Imagine you and your friend are talking on the phone.

You ask your friend a question, and your friend gives you an answer.

An API works in a similar way.

* You (the client) send a request.
* The API receives the request.
* The server processes the request.
* The API sends back a response.

For example, when you open a weather app, the app asks a weather service for today's weather using an API. The weather service sends the information back, and the app displays it on your screen.

In simple terms, an API is a messenger that helps different applications communicate with each other.
