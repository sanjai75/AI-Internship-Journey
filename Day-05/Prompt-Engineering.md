# Prompt Engineering Techniques

## 1. Zero-Shot Prompting

Zero-Shot Prompting means asking the AI a question directly without providing any examples.

### Example Prompt

```text
Explain APIs in simple terms.
```

### Understanding

The AI answers based on its existing knowledge without needing any sample examples.

---

## 2. Few-Shot Prompting

Few-Shot Prompting means providing one or more examples before asking the actual question.

### Example Prompt

```text
Example 1:
Python is a programming language.

Example 2:
Java is a programming language.

Now explain:
C++ is what type of language?
```

### Understanding

The examples help the AI understand the expected format and improve the quality of the response.

---

## 3. Role-Based Prompting

Role-Based Prompting means assigning a role or profession to the AI before asking a question.

### Example Prompt

```text
Act as a Senior Python Developer and explain decorators.
```

### Understanding

The AI responds from the perspective of the assigned role, often providing more relevant and detailed answers.

---

## 4. Step-by-Step Prompting

Step-by-Step Prompting asks the AI to explain a process in a sequence of steps.

### Example Prompt

```text
Explain how an API request works step by step.
```

### Understanding

The AI breaks down complex topics into smaller and easier-to-understand steps.

---

## Conclusion

Prompt Engineering helps users get better responses from AI systems. Common techniques include Zero-Shot Prompting, Few-Shot Prompting, Role-Based Prompting, and Step-by-Step Prompting.
