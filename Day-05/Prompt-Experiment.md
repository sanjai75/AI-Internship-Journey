# Prompt Experiment

## Topic Chosen: APIs

### 1. Basic Prompt

**Prompt:**

```text
What is an API?
```

**Response Summary:**
An API is a tool that allows different applications to communicate with each other.

---

### 2. Detailed Prompt

**Prompt:**

```text
Explain what an API is, how it works, and provide a real-world example suitable for beginners.
```

**Response Summary:**
The response explained the definition of an API, how requests and responses work, and included examples such as weather apps and payment gateways.

---

### 3. Role-Based Prompt

**Prompt:**

```text
Act as a Senior Software Engineer and explain APIs to a beginner.
```

**Response Summary:**
The response was more professional and detailed. It included technical concepts, practical examples, and industry usage.

---

### 4. Step-by-Step Prompt

**Prompt:**

```text
Explain how an API request works step by step.
```

**Response Summary:**
The response described the complete process:

1. Client sends a request.
2. Server receives the request.
3. Server processes the request.
4. Server sends a response.
5. Client displays the result.

---

## Comparison

| Prompt Type         | Quality   | Accuracy | Detail Level |
| ------------------- | --------- | -------- | ------------ |
| Basic Prompt        | Good      | High     | Low          |
| Detailed Prompt     | Very Good | High     | Medium       |
| Role-Based Prompt   | Excellent | High     | High         |
| Step-by-Step Prompt | Excellent | High     | High         |

---

## Observations

1. Basic prompts provide short and direct answers.
2. Detailed prompts generate more informative responses.
3. Role-based prompts produce expert-level explanations.
4. Step-by-step prompts make complex topics easier to understand.
5. Better prompts generally produce better and more useful results.

## Conclusion

The quality of AI responses depends heavily on the quality of the prompt. Detailed and structured prompts provide more accurate, useful, and informative answers than simple prompts.
