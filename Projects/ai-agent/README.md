# AI Agent

## Overview

This project demonstrates a simple Rule-Based AI Agent built using Python.

## Features

- Calculator Tool
- Current Date & Time Tool
- Random Quote Generator
- Mock Weather Lookup
- Rule-Based Decision Making

## Files

- tools.py - Contains all Python tools.
- agent.py - Rule-Based AI Agent.

## Run

```bash
python agent.py
```

User: What time is it?
