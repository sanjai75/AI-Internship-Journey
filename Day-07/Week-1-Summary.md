# Week 1 Learning Summary

During this week, I learned the fundamentals of Python programming. I understood variables, data types, input and output, conditional statements, loops, functions, and solved basic programming exercises. These topics helped me improve my problem-solving skills and understand how Python programs work.

I also learned about APIs and how applications communicate with each other using HTTP methods like GET and POST. I understood REST APIs, status codes, and how JSON is used to exchange data between applications. I practiced using Postman to test APIs and wrote Python programs to fetch data from public APIs.

Another important topic was Git and GitHub. I learned how to create repositories, track changes, commit code, and push projects to GitHub. This helped me understand version control and how developers collaborate on software projects.

I studied Large Language Models (LLMs), Prompt Engineering, and AI APIs. I learned how prompts affect AI responses and practiced different prompting techniques such as Zero-Shot, Few-Shot, Role-Based, and Step-by-Step prompting. I also understood concepts like tokens, context windows, and temperature.

Finally, I built a simple AI chatbot using Python and an AI API. The chatbot accepts user questions, sends them to an AI model, and displays the response. Through this project, I learned how AI applications communicate with LLMs using APIs and how to manage API keys securely.

Overall, this week gave me a strong foundation in Python, APIs, Git, and AI development. I also gained practical experience by completing hands-on coding exercises and mini projects.
